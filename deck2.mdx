## Gatsby Vision

Vamos começar então pela filosofia e visao do projeto.

Construct new, higher-level web building blocks

Construir uma linguagem de alto nível para a web,
no fim do dia o que o browser entende é CSS HTML e JS
mas não precisamos escrever nossos sites nestas linguagens
assim como nao precisamos escrever codigo de máquina / assembly.

Estamos num evento de React, alguém tem saudades dos tempos de antes do react?
Manipulação de DOM? Escrever HTML na unha? CSS global? Template languages
com apenas um subset de lógica e não o Javascript todo.

</Notes>


“You get simplicity by finding slightly more sophisticated building blocks“.
— Alan Kay

## Gatsby Vision

O exemplo que o Alan Kay citado no slide anterior da nesta apresentacao que influenciou
a filosofia do Gatsby, é o de como os cientistas do seculo 16 tetavam descrever as orbitas
dos planetas assumindo que elas eram circulos.

E o Kepler ter o insight de tentar descrever estas orbitas usando elipses ao inves de ciruclos
os modelos ficaram dramaticamente mais simples.

Antes de Kepler outros podem ter pensado nisso e desistido da ideia pois pessoas muito mais
inteligentes certamente "ja devem ter tentado".

From ancient times to the 1600s, people believed that the planets’ orbits must be perfect circles.
This false belief introduced a ton of complexity into their models.
This complexity persisted until Kepler came along with the stunning insight (hindsight sarcasm)
that no, orbits were elliptical not circular. With this new insight, models of planetary motion became dramatically simpler.
